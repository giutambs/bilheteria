2. Banco de dados (Firebase, gratuito)

1. Crie um projeto grátis em console.firebase.google.com (botão "Add project")
2. No menu lateral: Build → Firestore Database → Create database → escolha "Start in test mode"
3. Na aba Rules do Firestore, cole isto e clique em Publish:
allow read, write: if true; (dentro do bloco match /bilheteria_profiles/{doc} — substitua a regra padrão)
4. Vá em ⚙ Project settings → General → Your apps → clique no ícone </> (Web) → registre um app
5. Copie o objeto firebaseConfig inteiro e cole abaixo
